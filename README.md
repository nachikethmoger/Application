# chat-app-socket-io

### DESCRIPTION

It's a real time chat application where people connected to same host or socket  can exchanges the messages b/w them in the real-time.

### TECHNOLOGY USE

* Socket.IO enables real-time bidirectional event-based communication.
* Node JS
